## 开发规范

### 代码规范
- **Python**: 遵循PEP 8，使用Black格式化
- **TypeScript/JavaScript**: 使用ESLint和Prettier
- **提交信息**: 使用Conventional Commits格式

### 文档规范
- **文件格式**: Markdown (.md)
- **图片格式**: PNG, JPG, SVG
- **命名规范**: 使用连字符分隔 (kebab-case)
