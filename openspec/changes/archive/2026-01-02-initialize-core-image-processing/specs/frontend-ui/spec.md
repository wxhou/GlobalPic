# Spec: Frontend User Interface

## ADDED Requirements

### Requirement: 图像上传界面

**Scenario**:
- 用户拖拽图片到上传区域
- 支持JPG/PNG/WEBP格式，最大20MB
- 最多同时上传9张图片
- 显示上传进度条
- 上传前预览功能

### Requirement: 功能选择界面

**Scenario**:
- 功能复选框：智能去文字、场景重绘、主体分割、尺寸适配
- 背景风格网格选择：极简纯色、现代家居、商业环境、自然光线
- 风格预览缩略图展示
- 平台尺寸预设选择
- 参数调节滑块（亮度、饱和度、风格强度）

### Requirement: 处理进度界面

**Scenario**:
- 实时进度条显示（0-100%）
- 处理步骤状态：正在分析图片... (10%) → 正在检测文字... (25%) → 正在分割主体... (50%) → 正在生成背景... (75%) → 正在优化图片... (90%)
- 预估完成时间倒计时
- 任务取消按钮
- WebSocket实时更新

### Requirement: 结果展示界面

**Scenario**:
- 4张不同风格候选图片展示
- 原图vs处理后对比滑动条
- 单张图片放大查看
- 批量下载功能
- 社交分享按钮
- 重新处理选项

### Requirement: 用户认证界面

**Scenario**:
- 响应式登录表单
- 邮箱和密码输入验证
- "记住我"功能
- 忘记密码链接
- 注册页面邮箱验证

## 技术实现
- React 18 + TypeScript
- Tailwind CSS响应式设计
- React Query状态管理
- WebSocket实时通信
- react-dropzone文件上传

## 验收标准
- 页面加载时间 < 2秒
- 交互响应时间 < 100ms
- 移动端兼容性良好
- 无障碍访问支持
